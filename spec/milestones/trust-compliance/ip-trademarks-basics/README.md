# IP and trademarks basics

**Stage:** Pre-seed · **Category:** Trust & compliance · **[conditional]**

> Run a name search before you build any brand equity. File a trademark or provisional patent when the name is locked and the IP is core to the business.

---

{% hint style="info" %}
**Conditional.** This milestone applies when your company name, brand, or core technology is a meaningful competitive asset. If you're still pre-name or pre-product, the main action is IP assignment — ensuring all work is formally owned by the company, not the founders personally.
{% endhint %}

## What you are aiming for

All pre-incorporation IP formally assigned to the company, a name that has been cleared for use, and a trademark or provisional patent filed where the business depends on it — before you build significant brand equity or product value on an unprotected foundation.

## What to focus on

- **IP assignment first and always** — any code, designs, or inventions created before incorporation must be formally assigned to the company; this is the single most common diligence flag at pre-seed and it is free to fix now
- **Name search before launch** — a trademark conflict discovered after you've built brand equity is expensive; run the search before printing business cards
- **Provisional patent for core technology** — if your competitive advantage depends on a novel technical method, a provisional patent costs ~$1,500–$3,000 with a lawyer and buys 12 months to file the full application
- **Trade secrets for everything else** — most startup IP is better protected as a trade secret (through NDAs, access controls, and employment agreements) than through patents

## Deprioritize for now

- Patenting everything — most startup IP isn't patentable, and patents are expensive and slow; be selective
- International filings before you have customers in those markets
- Elaborate IP strategy documents — a 30-minute conversation with a startup IP lawyer is worth more

## IP assignment — the non-negotiable

Before you do anything else:

1. **Incorporate** — IP can only be assigned to a legal entity; this requires the company to exist
2. **Sign IP assignment agreements** — every founder, contractor, and early employee must sign a Proprietary Information and Inventions Assignment Agreement (PIIA) assigning all relevant work to the company
3. **List pre-incorporation work explicitly** — vague "all prior work" clauses can be challenged; list the specific projects, repos, or inventions that are being assigned

If you used contractors before incorporating, get a signed IP assignment from each of them now. This is a diligence requirement at every financing round.

## Trademark basics

A trademark protects a name, logo, or slogan used in commerce.

**When to file:**
- When the company name is final and you're using it publicly
- When a brand name is central to your go-to-market (consumer brands, developer tools with distinctive names)
- Before significant marketing spend on the name

**How to start:**
1. Run a USPTO trademark search at [tmsearch.uspto.gov](https://tmsearch.uspto.gov) — free, takes 30 minutes
2. Search Google and common domains for conflicting names
3. If clear, file a trademark application ($250–$350 per class, per mark) — or use a lawyer ($1,000–$2,000 all-in)
4. Expect 8–12 months for registration; you have protection from the filing date

**Classes matter:** trademarks are filed by goods/services class. Filing in the wrong class offers no protection in your actual market.

## Provisional patents

A provisional patent application establishes a priority date for an invention and gives you 12 months to file the full (non-provisional) application.

**When it's worth it:**
- Your core product relies on a novel technical method that competitors could copy
- You're in a patent-heavy industry (biotech, medtech, hardware)
- You're planning to raise institutional capital and investors in your space expect IP protection

**When it's not:**
- Your advantage is execution speed, distribution, or network effects — not a novel technical method
- You're a pure software company without a defensible algorithmic or process innovation

Cost: $1,500–$3,000 with a patent attorney for a well-drafted provisional.

## Quality levels

| Level | What it looks like |
|-------|--------------------|
| **L0 — No action taken** | No IP assignment; name not searched; no filings |
| **L1 — Assignment done** | All founders and contractors have signed IP assignment agreements; company owns all prior work |
| **L2 — Name cleared** | Trademark search completed; no material conflicts found; company using the name with confidence |
| **L3 — Filed where it matters** | Trademark filed for company name and/or key brand assets; provisional patent filed if core technology warrants it |

## How this connects to other goals

- **Co-founder alignment** — IP assignment is part of the co-founder agreement; cover it in [co-founder alignment](../../hiring-team/co-founder-alignment/README.md) so it doesn't get missed
- **Company financial backbone** — IP assignments require a legal entity to assign to; [company financial backbone](../../operations-back-office/company-financial-backbone/README.md) must be in place first
- **Early financing rounds** — investors will check IP assignment in diligence; missing assignments are a condition to close that delays the round

{% hint style="success" %}
**Done when:** Every founder and contractor has signed an IP assignment agreement, the company name has been searched, and any trademarks or provisional patents the business depends on have been filed.
{% endhint %}
